## Multipurpose Game Box ##
----------
The Multipurpose Game Box (aka MGB) can be programmed to perform multiples functions availaible in its firmware.  
It can be used as needed as :  
 - Medic Box
 - Ammo Box
 - Flag Station
 - Respawn Station
 - Team Respawn Station
 - Hazard
 - Mine
 - Traps
 - Stasis
 - Stun
 - etc.
